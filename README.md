# 三国数学 (Three Kingdoms Math)

一个结合《三国演义》历史背景与中学数学考纲的闯关式学习应用。

## 项目特色

- **历史与数学融合**：从“黄巾之乱”到“三分归一”，在经典战役中解决数学难题。
- **全考纲覆盖**：涵盖 7 年级至 12 年级的数学知识点，包括代数、几何、函数、概率、统计及微积分。
- **角色系统**：选择曹操、刘备或孙权，利用不同的武将技能辅助解题。
- **锦囊妙计**：内置“兵法核心”与“军师叮嘱”，提供公式推导与解题技巧。
- **双语支持**：支持中英文切换。

## 技术栈

- **前端**：React + TypeScript + Tailwind CSS
- **动画**：Framer Motion
- **数学公式渲染**：KaTeX
- **后端 (可选)**：Firebase (Firestore & Auth)

## 如何运行

1. 克隆仓库：
   ```bash
   git clone <your-repo-url>
   ```
2. 安装依赖：
   ```bash
   npm install
   ```
3. 启动开发服务器：
   ```bash
   npm run dev
   ```
